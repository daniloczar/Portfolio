Stroop Game

This project is a simple web-based game designed to test your cognitive abilities using the Stroop effect. The Stroop effect is a psychological phenomenon where the time it takes to name the color of a word is longer when the word itself is the name of a different color. In this game, you'll be challenged to quickly identify the color of the text displayed on the screen.
